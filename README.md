# javamyadmin
java版本的phpmyadmin

# 典型使用场景
- mysql服务器未开放远程连接，但希望能够进行远程管理
- 避免给出服务器的账号密码用于navicat等桌面管理工具通过ssh隧道进行连接

# 技术选型
- spring boot
- mybatis plus
- vue
- element ui
